# Running (Frontend)

## Prerequisites
- Windows 10/11
- Node.js (LTS recommended) + npm

## Run
From this folder, run:
- `RUN_FRONTEND.cmd`

Or manually:
- `npm install`
- `npm run dev`

## URL
- App: http://localhost:5173/

## Notes
If you see a PowerShell error about `npm.ps1` execution policy, run via CMD (the provided `RUN_FRONTEND.cmd` does this).
